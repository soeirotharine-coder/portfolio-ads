# Desenvolvimento Web

Conteúdos e atividades desenvolvidos na disciplina de Desenvolvimento Web,
utilizando HTML5, CSS, JavaScript e introdução ao PHP.

## Conteúdos estudados
- Estrutura básica de páginas HTML5
- Estilização com CSS
- Introdução ao JavaScript
- Noções iniciais de PHP
- Páginas web estáticas e dinâmicas

## Materiais
- Exercícios práticos
- Trabalhos acadêmicos
- Exemplos de páginas web simples
